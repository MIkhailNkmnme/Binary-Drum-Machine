## 🌌 Key Features & Mechanics

*   **🎯 3D Orbital Astrolabe (Radar View):** Instantly fold flat 2D binary grids into concentric polar coordinates. Dynamic orbit spin speeds can run in opposite directions to create complex moiré patterns and visual interference.
*   **🔊 Audio-Reactive VJ Core:** Real-time Web Audio API frequency analysis. Sub-bass frequencies drive the scale/zoom ("Breathing") of the grid, while overall track amplitude modulates the intensity of the Neon glow.
*   **🤖 AI Auto-Pilot (DJ Mode):** An autonomous algorithm monitors the incoming audio track. During musical build-ups, breakdowns, or drops, it automatically teleports through the "Treasure Map" scenes, shifting colors and cellular automata rules dynamically.
*   **🧬 Permanent XOR Mutations (Animatrix):** Run live horizontal wave anomalies that permanently mutate the underlying grid state, leaving a complex cryptographic footprint that never repeats.
*   **🛕 Emergent Geometry Engines:** Built-in fast generators for Classic Wolfram Cellular Automata (Rules 90, 30, 110, 18, 150), 3D Quantum Funnel, diagonal isometric meshes, and the unique Ziggurat structure.
*   **🫁 Breathing Zoom:** Smooth mathematical sine-wave scaling that brings the matrix closer to the viewer and recedes, creating a deep wormhole illusion.

*   **🧘 Immersive Zen Mode:** Double-click anywhere on the canvas to completely hide the UI panels and floating buttons. Only the pure, glowing code remains. Double-click again to return to the controls.
*   **🖱️ Free Pan & Zoom:** Left-click and drag to move the camera through space. Scroll the mouse wheel to scale the font size dynamically.
